## Descrição  

Um aplicativo simples e rápido criado para auxiliar no trabalho. Ele lê uma planilha base e insere os dados em um banco de dados.  
O projeto é direto ao ponto e fácil de usar.  

## Funcionalidade Principal  

O app permite importar produtos de arquivos `.xlsx` diretamente para o banco de dados, criando grupos e subgrupos automaticamente se necessário. Ele também permite limpar tabelas de produtos existentes com segurança.  

## Utilização  

1. Clone este repositório:  
  ```bash
  git clone https://github.com/Ntzzn-Dev/Excel2DB.git
  ```
2. Entre na pasta do projeto e instale as dependências:
  ```bash
  flutter pub get
  ```
3.  
  - Execute no Windows:
    ```bash
    flutter run -d windows
    ```
  - Execute no Linux:
    ```bash
    flutter run -d linux
    ```
## Tecnologias
